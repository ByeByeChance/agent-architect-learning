# LLM 的能力与边界 — 扩展参考

## 补充：Scaling Law

"大力出奇迹"的数学依据。OpenAI 2020 年论文发现，模型性能与三个因素呈幂律关系：
- 模型参数量（N）：越大越好
- 训练数据量（D）：越多越好  
- 训练计算量（C）：越强越好

三者需要均衡增长，否则会"欠拟合"或"过拟合"。

## 补充：涌现能力 (Emergent Abilities)

当模型规模超过某个阈值后，突然出现训练目标中没有明确教过的能力——比如数学推理、代码理解、角色扮演。小模型完全不会，大模型突然就会了。目前仍在研究中。

## 补充：In-Context Learning

LLM 不需要微调就能通过 prompt 中的几个例子学会新任务。这是 Transformer 架构的一个意外惊喜——训练时没有专门设计这个能力，但它自然出现了。
